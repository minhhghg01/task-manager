# Hệ Thống Quản Lý Công Việc Bệnh Viện

Hệ thống quản lý giao việc (Task Management) dành cho bệnh viện quy mô 1000 nhân viên, hỗ trợ phân quyền theo ma trận (Ban Giám Đốc, Trưởng khoa, Nhân viên) và thông báo thời gian thực.

## Công Nghệ Sử Dụng
- **Backend:** Node.js (Express)
- **Database:** SQLite (với Sequelize ORM)
- **Frontend:** EJS Template Engine, Bootstrap 5
- **Real-time:** Socket.io
